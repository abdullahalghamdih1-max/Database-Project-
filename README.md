This is a Delivery System database. It tracks how products get from the warehouse to the customer.
The 3 Main Parts:
1. Sales: Customers make Payments and place Orders for Products.
2. Logistics: Drivers use Cars to work specific Shifts.
3. Operations: The Delivery table connects everything—linking the Order, the Product, and the Driver to get the job done.
Key Goal:
To know who bought what, when they paid, and which driver delivered it using which car.
